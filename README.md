# Health impacts and socioeconomic inequalities of future outdoor air pollution in Europe

This is the code used for the analyses and figures of Rodés-Bachs, C., Sampedro, J., Van de Ven, Dirk-Jan & Amich. M. *Health impacts and socioeconomic inequalities of future outdoor air pollution in Europe*.

## Usage

The code is written in [R](https://www.r-project.org/) and uses the data available at XXX-ZenodoLink-XXX. The data processing is detailed in `R/data_preprocess.R`, although the raw files are not available due to its size. To reproduce the analysis code and the figures run `R/analysis_clean.R`
